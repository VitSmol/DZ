# DZ6
JS add Objects
Implement function that can “add” objects 

example:  
var a = {x: 1}; 
var b = {x: 2, y: 2}; 
add(a, b) // {x: 3, y: 2} 
add(a, b, a) // {x: 4, y: 2}
